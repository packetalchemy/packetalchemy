# Understanding 5G Core Architecture

The 5G Core (5GC) is the heart of modern mobile networks. It introduces a Service-Based Architecture (SBA) that separates network functions into modular services.

## AMF

The Access and Mobility Management Function (AMF) handles registration, authentication, and mobility management.

## SMF

The Session Management Function (SMF) is responsible for PDU session establishment, modification, and release.

## UPF

The User Plane Function (UPF) forwards user traffic and applies QoS and routing policies.
